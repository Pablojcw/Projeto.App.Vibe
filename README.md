# Projeto Vibe

Repositório de trabalho da faculdade para o projeto Vibe.

## Estrutura

- `public/` — arquivos estáticos que serão servidos diretamente.
- `src/` — código-fonte organizável em subpastas como `css`, `js`, `pages` etc.

## Como usar

1. Abra em um servidor (VS Code Live Server, Python `http.server`, etc.).
2. Navegue até `public/index.html`.

### Notas

- Os arquivos HTML originais estão em `src/pages`.
- Estilos estão em `src/css` e scripts em `src/js`.
